INSTALLATION
------------
1. Extract ALL files from the .zip
2. Windows: Run Disruption.exe
   Mac: Open Disruption.app (right-click → Open if Gatekeeper blocks it)
   Linux: Run ./Disruption.x86_64 (may need: chmod +x Disruption.x86_64)

CONTROLS
--------
Movement:
  WASD or Arrow Keys - Move in 8 directions
  Shift - Sprint
  Space - Jump/Hop
  Left Ctrl - Roll/Dodge
  Q - Grapple (when near grapple points)

Interaction:
  E - Interact with objects/NPCs
  ESC - Pause menu
  Enter - Confirm
  ESC - Cancel

Combat (if implemented in this build):
  Mouse Click - Select action/target
  Tab - Cycle targets
  1-4 - Quick select abilities
